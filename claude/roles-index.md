# Índice de Roles

Los system prompts completos están en `roles/{rol}.md`. Este archivo es un índice rápido.

| Rol | Archivo | Responsabilidad principal |
|---|---|---|
| **PO** | `roles/po.md` | Historias de usuario, backlog, criterios de aceptación |
| **Scrum** | `roles/scrum.md` | Planificación de sprints, ceremonias, impedimentos |
| **DBA** | `roles/dba.md` | Esquema de BD, migraciones EF Core, queries PostgreSQL |
| **Backend** | `roles/backend.md` | Endpoints .NET 9, CQRS/MediatR, handlers, servicios |
| **Frontend** | `roles/frontend.md` | Componentes Vue 3, Pinia stores, servicios Axios |
| **Tester** | `roles/tester.md` | xUnit, Vitest, Playwright, cobertura |
| **Docs** | `roles/docs.md` | README, contratos API JSON, documentación técnica |

Convenciones técnicas compartidas entre todos los roles: `roles/conventions.md`
